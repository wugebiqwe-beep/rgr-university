# Інформаційна система «Університет»

## Опис проєкту
Даний проєкт є навчальною інформаційною системою «Університет», розробленою в межах розрахунково-графічної роботи з дисципліни «Організація баз даних».

Система призначена для обліку основних структурних елементів університету: факультетів, кафедр, груп, студентів, викладачів та дисциплін. Веб-застосунок реалізований з використанням PHP та СУБД MySQL і підтримує повноцінну взаємодію з базою даних через графічний інтерфейс.

---

## Функціональні можливості
- перегляд факультетів, кафедр, груп, викладачів та дисциплін;
- повний CRUD (створення, читання, редагування, видалення) для сутності **«Студенти»**;
- використання звʼязків між таблицями (foreign key);
- валідація введених даних;
- захист від некоректного вводу користувача.

---

## Використані технології
- **PHP** — серверна мова програмування;
- **MySQL** — система керування базами даних;
- **HTML / CSS** — побудова інтерфейсу користувача;
- **XAMPP** — локальне серверне середовище;
- **phpMyAdmin** — адміністрування бази даних.

Вибір даних технологій зумовлений їх поширеністю, простотою використання та доцільністю для створення веб-інформаційних систем з БД.

---

## Структура бази даних
База даних містить такі основні таблиці:
- `faculties` — факультети;
- `departments` — кафедри;
- `groups` — академічні групи;
- `students` — студенти;
- `teachers` — викладачі;
- `subjects` — дисципліни.

Між таблицями реалізовані логічні звʼязки типу **один-до-багатьох**.

 ## Інструкція з запуску
1. Встановити локальний сервер **XAMPP**.
2. Скопіювати папку `rgr_university` до директорії `htdocs`.
3. Запустити **Apache** та **MySQL**.
4. Відкрити **phpMyAdmin** та створити базу даних `university_db`.
5. Імпортувати файл `sql/01_schema.sql`.
6. Імпортувати файл `sql/02_data.sql`.
7. Відкрити в браузері: http://localhost/rgr_university/
